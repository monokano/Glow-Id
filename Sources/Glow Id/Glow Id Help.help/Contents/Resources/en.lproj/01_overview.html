About Glow Id

What Glow Id Does

InDesign files cannot be opened in older versions of InDesign. Furthermore, opening a file in a newer version than the one used to create it can cause layout changes. To avoid trouble, you need to accurately identify the version a file was created in and open it with the same version of InDesign.

Glow Id reads the creation version recorded inside each file and automatically selects the matching version of InDesign to open it with.

Glow Id is especially useful when multiple versions of InDesign are installed on your Mac.

Supported File Formats

InDesign Document (.indd)
Standard InDesign native format.
InDesign Template (.indt)
InDesign template format.
InDesign Book (.indb)
A collection of InDesign documents managed as a single book.
InDesign Library (.indl)
A library of reusable InDesign objects.
IDML File (.idml)
InDesign Markup Language format, used for opening documents in older versions of InDesign.
Assume that layout changes may occur when working with IDML files.
Note File type is determined from the file's content, regardless of the file extension. Documents and templates differ only in their extension, so after determining file type from the content, the extension is used to distinguish between them.
