Notification Window

The Notification Window appears when no matching version of InDesign is found, when the file requires manual review before opening, or when an IDML file is opened.

File Information

Icon

Shows the icon of the dropped file. A warning badge may appear above the icon, and a kind badge may appear to its left.

Badge

Displays a warning message or file type label when needed.

Warning badge (above the icon)
Reports the file's status. Shown when no matching version of InDesign is installed, or when another version of InDesign is already running.
Kind badge (to the left of the icon, red)
Shown when the file extension does not match the file content, or when the file cannot be identified as an InDesign file.

Filename

The name of the dropped file.

File Type

The file type as determined from the file's content.
Example: InDesign Document (.indd)

Creation Version

The InDesign version that created the file, as read from the file's internal data, displayed as an app name.
Example: InDesign 2025 (20.5)

Version Matching

How the file's creation version is compared against installed InDesign apps depends on the file type.

  • Document (.indd) / Template (.indt) / Book (.indb): Matched by major.minor version
  • Library (.indl): Matched by major.minor version by default. For CS6 (8) through CC 2018 (13), matched by major version only
  • IDML (.idml): No version matching. The Notification Window always appears so you can choose which version to open

App List

Lists all versions of InDesign installed on your Mac, sorted from oldest to newest.

Blue dot (●)
Indicates the app is currently running.
App name
The name of the installed app. Beta and Prerelease builds are always greyed out to prevent accidental use in production. Versions of InDesign older than the file's creation version are also greyed out.

Opening a File from the List

Double-click a row to open the file with that version of InDesign. Alternatively, select a row and click the "Open" button. Double-clicking a greyed-out row shows an "Open" menu item that lets you force-open the file.

Buttons

Open
Opens the file with the selected app.
Cancel
Closes the window without opening the file.
Quit
Quits the currently running InDesign. The blue dot disappears once InDesign has actually terminated.

Checkbox

Open with newer version
When checked, versions of InDesign newer than the creation version become available for opening the file. When unchecked, newer version rows are greyed out. This setting is remembered across sessions.
