Glow Id について

何をするアプリか

Adobe InDesign のファイルは、下位バージョンの InDesign で開けません。さらに、作成バージョンと異なる上位バージョンで開くと、レイアウトが変化してしまうことがあります。トラブルを回避するには、ファイルの作成バージョンを正確に把握し、同じバージョンの InDesign で開く必要があります。

Glow Id は、ファイルの内部に記録されている作成バージョンを読み取り、同じバージョンの InDesign を選んで自動的に開きます。

複数バージョンの InDesign が Mac にインストールされている環境で特に威力を発揮します。

対応ファイル形式

InDesign ドキュメント（.indd）
標準的な InDesign ネイティブ形式。
InDesign テンプレート（.indt）
InDesign テンプレート形式。
InDesign ブック（.indb）
複数の InDesign ドキュメントをまとめたブック形式。
InDesign ライブラリ（.indl）
よく使うオブジェクトを収納するライブラリ形式。
IDML ファイル（.idml）
InDesign Markup Language 形式。下位バージョンで開くために使われます。
レイアウトが変化するかもしれないことを前提に運用するものです。
注意 拡張子に関わらず、ファイルの内容をもとに種類を判定します。ドキュメントとテンプレートはそもそも拡張子が異なるだけなので、ファイルの内容を判定した後に拡張子で区別します。
