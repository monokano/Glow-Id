Glow Id ヘルプ › 使い方

使い方

ファイルを開く

方法 1：ダブルクリック（推奨）

Glow Id を一度起動すると、.indd / .indt / .indb / .indl / .idml ファイルが Glow Id に自動的に関連付けられます。以降はファイルをダブルクリックするだけで Glow Id が起動し、バージョンを判定して適切な InDesign で開きます。

  1. .indd / .indt / .indb / .indl / .idml ファイルをダブルクリックします。
  2. Glow Id が起動し、ファイルのバージョンを判定します。
  3. バージョンが一致する InDesign が見つかれば自動的に開きます。見つからない場合や、確認が必要な場合は通知ウインドウが表示されます。
ヒント ダブルクリックで Glow Id が起動しない場合は、設定の「起動時に関連付けをする」がオフになっている可能性があります。オンにするだけで Glow Id と関連付けされます。

方法 2：Dock アイコンにドロップ

  1. Dock に表示されている Glow Id のアイコンに、ファイルをドラッグ＆ドロップします。
  2. ファイルのバージョンが判定され、一致する InDesign が自動的に起動します。
  3. バージョンが一致するアプリが見つからない場合や、確認が必要な場合は通知ウインドウが表示されます。

方法 3：メニューから開く

  1. メニューバーの「ファイル」→「開く…」（⌘O）をクリックします。
  2. ファイル選択ダイアログでファイルを選びます。複数ファイルを同時に選択できます。

IDML ファイルについて

IDML は異なる下位バージョンで開くためのファイル形式なので、常に通知ウインドウが表示されます。警告も表示されません。開くバージョンの InDesign を選択してください。

複数ファイルをまとめて処理する

複数のファイルを一度に開くと、1ファイルずつ順番に処理します。通知ウインドウが表示される場合は、1つのウインドウを閉じてから次のウインドウが表示されます。処理が終わってからまとめて InDesign で開きます。

⌘ キーを押しながら開く

バージョンが一致する InDesign が見つかった場合、通常は自動的に開きます。⌘（Command）キーを押しながらファイルを開くと、バージョンが一致していても通知ウインドウを表示します。

Glow Id は自動で終了する

ファイルを開いた後、Glow Id は自動的に終了します。InDesign の作業中に Glow Id を手動で終了する必要はありません。

ヒント 通知ウインドウの「終了」ボタンを使うと、現在起動中の InDesign を Glow Id から終了させることができます。

インストール後・アップデート後の初回起動

Glow Id をインストールした後、またはアップデートで差し替えた後は、必ず一度 Glow Id を起動してください。

アップデートで差し替えると .indd / .indt / .indb / .indl / .idml ファイルとの関連付けが外れることがあります。Glow Id を起動するだけで自動的に復元されます。

また、次のいずれかに該当するとき、起動時に InDesign のファイルアイコンを自動で取り込みます：

  • インストールされている InDesign の最上位バージョン（Beta・Prerelease 除く）が変わったとき
  • Glow Id 自体が新しいバージョンになったとき

アイコンの取り込みは、Glow Id が「アプリケーション」フォルダなど書き込み可能な場所から起動しているときに行われます。ダウンロードフォルダで展開した場所からそのまま起動していると取り込みできず、「アプリケーションフォルダへ移動してください」という案内を表示します。その場合は Glow Id を「アプリケーション」フォルダ（管理者でないアカウントでは、ホームフォルダ内の「アプリケーション」フォルダ）へ移動してから、起動し直してください。
